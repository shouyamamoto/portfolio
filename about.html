ME

山本 翔

1996年生まれ 23歳 兵庫県姫路市出身

私は理学療法士として、「その人らしさ」をモットーに仕事に励んできました。

「その人らしさ」を実現するため、身体や精神に障害を持ってしまった患者様に寄り添ったヒアリングや治療を行ってきました。

加えて、家族様や担当医、看護師など、患者様を取り巻く方々との専門的視点を持った情報共有を行い、チーム一丸となって患者様が満足のいく状態で御退院していただけるよう努めてきました。

これらの経験をいかし、クライアントに寄り添ったヒアリング・提案を行い、チームとしてニーズに答えられるようなデザイナーになりたいと思っています。

SKILL

  • HTML5

    適切なタグ、見やすいインデント、明確なクラス名で修正しやすいよう記述するようにしています。

  • CSS3

    Sassと連携し、運用・拡張がしやすいような設計を心がけています。

    Sassを利用することで、cssのみの記述よりも素早く・性格な設計を行えます。

  • Javascript

    jQueryで対応できない動きや複雑な動きが必要な場合には、javascriptで動きを実装します。

  • jQuery

    jQueryを用いることでサイト上に様々な動きを加えることができます。
    必要に応じてプラグインを用いることにより、javascriptよりも素早い機能実装が可能です。

  • Ruby

    Rubyを用いてログイン機能やお問い合わせ機能など、サーバーサイドでの動きの実装が可能です。

  • Ruby on Rails

    RubyのライブラリであるRuby on Railsを使用することでより素早く機能実装が可能です。

  • Github

    githubにてバージョン管理を行っています。複数人での開発は経験がありませんが、今後も学習を継続し,より保守性の高いコーディングを行ってまいります。

  • Photoshop

    Photoshopを使用し、写真加工や切り抜き、バナー作成が可能です。また、デザインカンプを作成することもできます。

  • Illustrator

    Illustratorを使用し、オブジェクトの作成やバナー作成が可能です。PhotoshopよりもIllustratorでのバナー作成が得意です。

  • Visual Studio Code

    多彩なプラグインが導入されているVsCodeにてコーディングを行っています。補完機能も充実しており、より素早く、正確なコードを書くことができます。

OTHER

前職での勉強会の主催

勤務していた病院にて、治療技術の勉強会が少ないことに違和感を感じ、毎月6回業務後や休日に院外での勉強会に参加していました。 そこで得た知識をアウトプットし、知識の定着化やコミュニケーション能力の向上、患者様への治療効果の貢献を図りました。
常に現状に満足せず、患者様がより良い状態で退院されるために職場での問題点を考えて計画・実行してきました。その結果、現状の課題の明確化、改善提案、実行力を身に着けることができました。
